# Blinkit_Sales_Dashboard
🛒 Blinkit Sales Dashboard (Power BI)

📌 Overview

* This project is a Power BI dashboard built to analyze Blinkit's sales performance across different outlets, item types, and locations. The goal is to extract meaningful insights and understand patterns in sales, customer ratings, and product demand.

🎯 Objectives

* Analyze overall sales performance
* Understand item category contribution
* Compare outlet performance by size and location
* Identify trends in outlet establishment over time
* Provide insights for better business decisions
  
📊 Key Metrics
* Total Sales: $1.20M
* Average Sales: $141
* Number of Items: 8523
* Average Rating: 3.9

📈 Key Insights

* Sales by Item Type
Fruits, Snacks, and Household items generate the highest sales
Seafood and Breakfast categories have the lowest contribution

*  Outlet Performance
* Tier 3 cities contribute the highest sales
* Tier 1 has comparatively lower sales

* Outlet Size Analysis
* Medium-sized outlets generate the highest revenue
* Smaller outlets also perform well but less than medium

* Outlet Type
* Supermarket Type1 contributes the highest sales
* Grocery stores have lower sales but higher item visibility

* Trend Analysis
* Sales increased significantly after 2016
* Stable growth observed in recent years

🛠️ Tools Used
* Power BI
* Data Cleaning & Transformation
* Data Visualization
  
* Conclusion
* The dashboard highlights that sales are highly influenced by outlet location, size, and item category. Medium-sized outlets in Tier 3 cities and high-demand categories like snacks and fruits are key contributors to revenue.
